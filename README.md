## Para empezar

1. Clona el repositorio

   ```sh
   git clone https://github.com/Afordin/hackafor-2.git
   ```

2. Instala los paquetes NPM
   ```bash
   npm install
   ```

3. Ejecuta el proyecto
   ```bash
   npm run dev
   ```
4. abre tu navegador
